支付收单参数列表

字段名 含义 类型 长度 是否可空 说明
merchantNo 商户号 string 18 否 由企业站统一分配。
shopId 门店号 string 100 是 商户门店ID
merchantUserId 商户的用户号 string 32 是 商户侧的用户ID
finCustomerId 金融会员ID string 32 是 京东金融的会员ID，某些情况下是必传的(比如钱包等产品)
orderId 商户订单号 string 32 否 由商户自动生成的订单唯一标识符。
orderType 订单业务类型 string 32 否 订单业务类型，参数见订单类型码表。
businessSceneId 业务场景ID string 32 是 业务场景ID，收银台报风控使用
businessRiskId 业务风控ID string 32 是 业务风控ID，支付报风控时使用，部分场景(如钱包充提转)下不能为空
mandatoryCheck 是否强制加验 Boolean - 是 本单是否强制加验，默认否
orderExpireTime 订单有效期截止时间 Date - 是 订单有效期截止时间
productType 订单产品类型 string 32 否 订单产品类型，参数见产品类型码表。
transactionAmount 收单金额 BigDecimal - 否 收单金额(用户实际支付金额)。当discount、shippingCost、tax的某一项不为空时，需要满足条件：transactionAmount = subtotal – discount + shippingCost + tax
subTotal 商品总价 BigDecimal - 否 商品总价(不包含营销等扣减项)。
discount 折扣 BigDecimal - 是 订单折扣、优惠等总金额
shippingCost 运费 BigDecimal - 是 订单运费金额
tax 税费 BigDecimal - 是 订单税费金额
orderCurrency 订单币种 string 3 否 支付币种，ISO标准3位字母。
orderCountry 订单国家 string 2 否 支付订单所属国家，ISO标准2位字母。
notifyUrl 异步通知地址 string 512 是 接收交易结果通知的地址(httppost)
returnUrl 同步返回地址 string 512 是 收银台支付成功后重定向的地址
orderItems 商品信息 List - 是 商品详细信息，非强校验，某些业务场景下不能为空，否则可能导致网关支付失败。
itemSku 商品SKU string 20 是 商品SKU。
itemCategory 商品品类 string 20 是 商品品类。
itemName 商品名称 string 50 是 商品名称。
itemPrice 商品单价 BigDecimal - 否 商品单价。
currency 币种 string 3 否 商品币种，ISO标准3位字母。
itemQuantity 购买商品数 Integer 5 否 商品数量。
itemLogo 商品图标 string - 是 商品logo。
shippingDetail 货运信息 ShippingDetail - 是 货运详细信息，非强校验，某些业务场景下不能为空，否则可能导致网关支付失败。
shippingFirstName 收货人名 string 64 否 -
shippingLastName 收货人姓 string 64 否 -
shippingPhone 收货人电话 string 21 是 电话和邮箱必传其一。
shippingEmail 收货人邮箱 string 64 是 电话和邮箱必传其一。
shippingCountry 收货人国家编码 string 2 否 ISO标准，2位字母。
shippingCountryName 收货人国家名称 string 127 是 -
shippingState 收货人州(省)编码 string 128 否 -
shippingStateName 收货认州(省)名称 string 128 是 -
shippingCity 收货人市 string 128 否 -
shippingStreet1 收货人街道 string 128 否 -
shippingStreet2 收货人门牌号 string 256 否 -
shippingPostal 收货人邮编 string 16 否 -

响应参数说明

字段名 二级字段 是否可空 名称 类型 说明
merchantId N 商家商户号 String 由企业站统一分配。
orderId N 商家订单号 String 请求时的商户订单号。
transactionNo N 交易单号 String 由金融生成交易单号唯一标识符。
cashierPcUrl N PC收银台url String 在线支付重定向至PC版收银台的地址，均以https开头。
cashierH5Url N H5收银台url String 在线支付重定向至H5版收银台的地址，均以https开头。
resultFlag N 是否成功 boolean 表示请求结果，成功为true,反之false
resultCode N 处理结果编码 String 表示请求结果，如果失败可以取改值
resultInfo N 处理结果描述 String 响应码文字说明，如果失败可以取改值

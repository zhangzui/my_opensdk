请求/响应公共参数列表

请求和响应报文格式均为json格式

字段名 变量名 类型 可空 说明
version 版本号 String N 1.0.0
charset 字符编码 String N 默认UTF-8
tradeType 交易类型 String 参考值见附录交易类型编码表（指交易接口功能，收单，查询，退款等）按参数列表组成JSON格式数据，然后使用AES加密。具体参数说明需要看接口参数列表
data 加密数据 String N AES加密之后的数据
merchantCode 商户号 String N 商家商户号
sign 数据签名 String N SHA256签名，签名顺序为version+charset+tradetype+data+merchantCode+privateKey+soltKeyN
